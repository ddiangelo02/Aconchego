# 🧶 Aconchegô

> *"Se você nos encontrou, é porque está em busca de um momento de aconchego, de pausa e de conexão... Conexão com alguém que você deixou para trás em algum momento... Você! Sinta-se à vontade para nos conhecer e assim, talvez, se conhecer também! É sempre um prazer poder se reencontrar."*

---

## 📖 Sobre o Projeto

O **Aconchegô** é um e-commerce voltado para a venda de produtos artesanais. Mais do que uma loja virtual, nosso espaço é um convite para o autocuidado, o respiro no meio da rotina agitada e a valorização do tempo, que se traduz no cuidado das peças feitas à mão. 

Nosso objetivo é conectar artesãos a pessoas que buscam produtos com significado, história e afeto, proporcionando uma experiência digital que seja tão acolhedora quanto o próprio nome da marca sugere.

---

## 🛠️ Tecnologias utilizadas

Para materializar esse espaço de aconchego, utilizamos as seguintes tecnologias:

*   **Front-end:** HTML5, CSS3, JavaScript e jQuery
*   **Back-end:** PHP e Laravel
*   **Banco de Dados:** MySQL
*   **Estilização:** TailwindCSS
*   **Design & Prototipagem:** Figma
